Hello Visitor!

My name is Tim and this my little demo website. The main purpose is to be able to play "Locator-IO" with others. The code for all these projects can be found at GitHub.

Locator.io

Play alone or against friends. Purpose of the game is to locate a city with only satellite imagery as a base. Get points based on distance to actual location. Most points win. Uses Websockets for realtime communication. Visually very basic. Not optimized for Phones, yet.

Watcher

Here, you can monitor the CPU and Memory Load and number of Goroutines (this page is build using GO) that are used by this Page/Server.

Chat

Here, you can create or join a Chatroom, and then chat in real time. Pretty basic, but no government tracking! I swear.
